👓 Specs Ecom – Lens Cart (Angular Online Spectacles Store)

📌 Project Overview

Specs Ecom (Lens Cart) is an online e-commerce web application built using Angular.
It allows users to browse, view, and purchase spectacles (spex) through a simple and responsive interface.

The project is structured using Angular components, routing, and modular architecture for scalability and maintainability.


---

🚀 Features

🛍️ Product listing for spectacles

🔍 Product details view

🛒 Add to cart functionality

❤️ Wishlist support (if implemented)

🔐 User-friendly navigation using Angular routing

📱 Responsive UI for mobile and desktop

⚡ Fast single-page application experience



---

🧱 Tech Stack

Frontend: Angular

Language: TypeScript, HTML, CSS

Routing: Angular Router

Architecture: Component-based modular structure

🔄 Routing (app-routing.module.ts)

The application uses Angular routing for navigation:

/home → Home page

/products → Product listing page

/product/:id → Product details page

/cart → Shopping cart page

⚙️ Installation & Setup

Follow these steps to run the project locally:

1️⃣ Clone the repository

git clone https://github.com/your-username/specs-ecom.git

2️⃣ Navigate to project folder

cd specs-ecom

3️⃣ Install dependencies

npm install

4️⃣ Run the application

ng serve

5️⃣ Open in browser

http://localhost:4200

🎯 How It Works

1. User lands on the Home page


2. Browses available spectacles in Product List


3. Clicks on a product to view details


4. Adds items to cart


5. Proceeds for checkout (if implemented)
